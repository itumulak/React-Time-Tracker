## Time Tracker

This is a simple app that monitors if a user's tracker is enabled or not.

This is built on `React.js` and `Next.js`.

### Additional packages used:
- Redux Toolkit
- Redux Persist
- Redux Thunk

### Running the app on development
```bash
npm run dev
# or
yarn dev
```
